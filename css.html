CSS DESCRIPTION

WHAT IS CSS

Refers to cascading style sheet language used to stylize elements written in a markup language such as HTML

WAYS CSS CAN BE INVERTED IN A WED PAGE

Inline - by using the style attribute inside HTML elements

With an external file that you link to in your web page

By creating a CSS block in the web page itself

DIFFERENT CSS SELECTORS

Simple selectors (select elements based on name, id, class)

Combinator selectors (select elements based on a specific relationship between them)

Pseudo-class selectors (select elements based on a certain state)
